Dynamically displaying the data for a single person

This page displays the single row from the people table that corresponds to the value of id in the URL, e.g.: /person?id=123 on the previous page.

app.py sends a tuple called person to this page and each item in person is displayed below.

There is no looping required on this page because it is only showing a single row of data.

The Edit link also uses the get method to pass the id value to /person/edit.
